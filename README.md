Desenvolvimento Front-End para Web
Repositório criado para registrar meus estudos, exercícios e projetos desenvolvidos durante a disciplina Desenvolvimento Front-End para Web, cursada no 1º semestre da faculdade de Ciência da Computação.

Sobre a Disciplina
A matéria introduz os conceitos básicos de criação de páginas web usando HTML, CSS e JavaScript.
O objetivo é aprender como montar a estrutura de um site, aplicar estilos e adicionar interatividade simples.

Conteúdos Estudados
HTML – Estrutura
Introdução às tags básicas
Cabeçalhos, parágrafos, listas e links
Imagens
Tabelas
Formulários simples
Estrutura semântica

CSS – Estilização
Seletores básicos
Cores, fontes e tamanhos
Margens, bordas e preenchimento
Display e posicionamento

JavaScript – Primeiros Passos
Variáveis e tipos de dados
Funções básicas
Eventos simples (clique, mudança, etc.)
Lógica para pequenos comportamentos interativos

Tecnologias Utilizadas

HTML5
CSS3
JavaScript básico
Visual Studio Code
Navegadores (para testes)



Objetivos de Aprendizagem no 1º Semestre

Entender como funciona a estrutura básica de um site
Aprender a estilizar páginas com CSS
Criar pequenas interações usando JavaScript
Começar a organizar projetos e pastas
Ganhar prática escrevendo código desde o zero

Minha Evolução

Este repositório vai mostrar minha evolução como estudante de TI, desde os primeiros exercícios até meus primeiros projetos completos no front-end.
